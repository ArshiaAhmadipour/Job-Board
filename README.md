# Django Job Board

A simple Django web application for managing and displaying job postings. Built as a beginner-friendly project to learn Django fundamentals.

## Features

- View all active job postings
- Job details include:
  - Job title
  - Company name
  - Salary
  - Job description
  - Active status (active/inactive)
- SQLite database for easy setup
- Django admin interface for managing jobs

## Technologies Used

- **Backend**: Django 4.x
- **Database**: SQLite
- **Frontend**: HTML with Django Templates
- **Language**: Python 3.x
